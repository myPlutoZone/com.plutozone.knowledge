# com.plutozone.knowledge.utility


- 파워포인트(.pptx), 엑셀(.xlsx) 등에서 이미지를 개별로 분리하는 방법
	- 확장자를 zip으로 변경
	- 해당 zip 파일을 압축 해제
	- 해제된 폴더중에 이미지 폴더에서 분리된 개별 이미지들을 확인
- 구글 검색
	- intitle: index of 키워드(예: intitle: index of hdd, intitle: index of mp3)
	- 키워드 site:plutozone.com(예: pluto site:plutozone.com)
		- 명시적으로 Disallow 처리되지 않은 모든 웹 페이지(링크가 없는 웹 페이지 포함) 검색
